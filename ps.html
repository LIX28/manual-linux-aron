Ejemplo de uso del comando ps

Mostrar procesos actuales:

ps

Mostrar todos los procesos:

ps aux

Buscar un proceso específico:

ps aux | grep nombre_proceso

Mostrar procesos con formato detallado:

ps -ef

EJEMPLOS DE ERRORES COMUNES

Error: olvidar opciones para ver todos los procesos

ps

Correcto:

ps aux

Error: buscar proceso sin usar grep

ps aux nombre

Correcto:

ps aux | grep nombre

Error: usar mal las opciones

ps -a -u -x (separado innecesariamente)

Correcto:

ps aux
